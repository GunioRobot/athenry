DESCRIPTION

Athenry will use the system configuration files you provide, to build what ever Gentoo Stage tarballs (currently stage3 - stage5) you want to make installing your own freshly rolled x86, amd64 for those computers at home and the friends that have been bugging you, or even if you need something for work like, sparc, PPC, ia64, all inside the safety of a single chroot, and using only Athenry.

For more information please refer to about.

REQUIREMENTS

Install the following gems:

gem sources -a http://gemcutter.org
sudo gem install configatron
sudo gem install commander
  • configatron
  • commander

INSTALLATION

See the QuickStart guide.

DOCUMENTATION

You can build the documentation by running

rake documentation:generate

Files generated will be stored inside meta/documentation
